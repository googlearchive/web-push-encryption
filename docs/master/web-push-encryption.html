web-push-encryption

web-push-encryption is a module that helps encrypt and / or send push messages over the web push protocol.

(static) encrypt(message, subscription, paddingLength) → {Object}

Encrypts a message such that it can be sent using the Web Push protocol. You can find out more about the various pieces:
  • https://tools.ietf.org/html/draft-ietf-httpbis-encryption-encoding
  • https://en.wikipedia.org/wiki/Elliptic_curve_Diffie%E2%80%93Hellman
  • https://tools.ietf.org/html/draft-ietf-webpush-encryption
Parameters:
Name Type Description
message String | Buffer The message to be sent
subscription Object A JavaScript Object representing a PushSubscription. The easiest way to get this from the browser to your backend is by running JSON.stringify(subscription) in a web pages JavaScript.
paddingLength number Number of bytes of padding to use
